# Wave Platformer (Froggy)

Welcome to **Wave Platformer**, a high-stakes, wave-based survival action game where precision movement meets intense combat! Survive relentless enemy waves, master complex combos, and weigh risk against reward in *The Devil's Bargain* before the sunrise inevitably takes its toll.

## 🌟 Key Features

### ⚔️ Intense Combat & Combos
Master a fluid combat system with distinct light and heavy attacks. String together moves to unleash devastating combos:
- **Flurry Rush:** Light, Light, Light, Light
- **Uppercut Launch:** Light, Light, Heavy
- **Dash Strike:** Dash, Light
- *...and many more including Ground Pounds, Meteor Slashes, and Whirlwinds!*

### 🕰️ The Sunrise Timer
You are always fighting against the clock. Every run features a real-time **15-minute Sunrise Timer**. The sky dynamically transitions from night to dawn—fail to escape before sunrise, and face an instant Game Over, adding to your Damnation.

### 😈 The Devil's Bargain & Progression
When a Sheet Fragment drops at the end of a wave, the game pauses for a critical choice:
1. **Take the Fragment** to escape and progress.
2. **Accept a Dark Power** to forfeit the fragment in exchange for run-altering passive buffs (like Lifesteal, Fire Fists, Blood Rage, or Hellfire).

Beware, dealing with the devil increases your **Damnation Meter**, which persists across runs!

### 🏃‍♂️ Advanced Locomotion
Navigate dynamically generated floating platforms with evolved movement tech:
- **iFrame Dashing:** Dodge incoming attacks with perfectly timed invincibility frames.
- **Dash Jump:** Execute high-velocity diagonal aerial dashes for unmatched verticality.

### 👹 Evolved AI & Bosses
Face off against strategic enemies who wind up melee attacks or predict your movement with fireballs (now restricted to mid-level enemies and bosses). Confront solitary Bosses like **The Abyss** in epic showdowns to unlock permanent *Special Attack Riffs* like the "Infernal Chord".

## 🎮 Controls

| Action | Keybinding |
| :--- | :--- |
| **Move** | `A` / `D` or `Left Arrow` / `Right Arrow` |
| **Jump** | `W` or `Up Arrow` or `Space` |
| **Crouch** | `S` or `Down Arrow` |
| **Light Attack** | `J` or `Left Mouse Click` |
| **Heavy Attack** | `K` or `Right Mouse Click` |
| **Dash / Dodge** | `Left Shift` |
| **Debug Menu** | ``` ` ``` (Tilde/Backtick) |

## 🛠️ How to Play / Run Locally

1. Clone or download the repository to your local machine.
2. Open the project folder (`froggy`).
3. You can run the game by directly open `index.html` in any modern web browser.
4. (Optional) For the best experience, host the folder using a local web server (e.g., using VSCode's Live Server extension, or `python -m http.server`).

## 🐛 Debug Options (For Testing)
Press the ``` ` ``` key to open the developer debug panel. Here you can tweak game speed, toggle God Mode, spawn specific waves, instantly Max Resources (Fragments, Echoes, Boss Defeats), or use the "Unlock All Levels" toggle.
